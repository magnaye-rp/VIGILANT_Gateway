Metrics Toolkit Guide

RX/TX Throughput

RX (Receive) and TX (Transmit) throughput measure the current network bandwidth usage in megabits per second (Mbps). Higher values indicate more network activity flowing through the gateway.

Packet Filtering

The gateway inspects and filters network packets based on content categories, keywords, and behavioral policies. Blocked requests are logged and can be reviewed in the Traffic Logs section.

Gateway Bandwidth Throttling

When devices exceed behavioral thresholds or are manually throttled, their bandwidth is limited to the configured rate (default: 256 Kbps). This helps manage network congestion and enforce usage policies.
